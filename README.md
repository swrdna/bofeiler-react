- Tailwind CSS
- pnpm

# UI Libs

- [shadcn/ui](https://ui.shadcn.com)

# How to start

Create env file

```
cp .env.example .env
```

Install dependencies

```
pnpm install
```

Run the project

```
pnpm dev
```
